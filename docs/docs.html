Snowtify guide

Documentation

Everything you need to install, customize, and restore Spotify from PowerShell.

01

Installation

Run PowerShell as a standard user.

Before you install: Use the desktop version of Spotify downloaded from spotify.com, not the Microsoft Store package. Open PowerShell as a standard user, not as administrator.

PowerShell
iwr -useb https://raw.githubusercontent.com/SnoW-099/snowtify/main/install.ps1 | iex

Snowtify reuses %APPDATA%\spicetify. Your existing themes, extensions, and settings remain available.

02

First run

Create a clean backup and apply your customization.

snowtify backup apply
03

Repair Snowtify

Restore your customization when Spotify updates or fails to start correctly.

snowtify repair
04

Update

Update Snowtify in place without uninstalling, reinstalling, or losing your configuration.

snowtify update

The updater downloads the latest Snowtify release, preserves your setup, and reapplies it to Spotify automatically.

Spotify shows a one-time notification and a top-bar update button only when a newer Snowtify version is available.

05

Marketplace

Install community themes and extensions from Spotify.

Snowtify preserves the internal Spicetify.* API and the compatibility alias required by Marketplace. The default Snowtify Frost theme appears under Marketplace > Installed, where you can remove it or replace it with any Marketplace theme.

06

Essential commands

A quick reference for everyday use.

CommandPurpose
snowtify --versionShows the installed Snowtify version.
snowtify applyApplies the current configuration.
snowtify repairRepairs and reapplies after an update.
snowtify updateUpdates Snowtify in place and preserves your setup.
snowtify restoreRestores Spotify to its original state.
snowtify configDisplays or changes the configuration.
07

Troubleshooting

Quick fixes to try before asking for help.

Snowtify disappeared after Spotify updated

Run snowtify repair. If Spotify changed significantly, run snowtify update first.

Spotify opens with a frozen interface

Temporarily disable the latest theme or extension you installed, then run snowtify apply.

PowerShell cannot find the snowtify command

Close PowerShell and open it again so Windows reloads your user PATH, then run snowtify --version.

08

Uninstall Snowtify

Restore Spotify and remove Snowtify safely.

Run this only if you want to remove Snowtify.

The official uninstaller restores Spotify before deleting any Snowtify files.

PowerShell
iwr -useb https://raw.githubusercontent.com/SnoW-099/snowtify/main/uninstall.ps1 | iex

Removed Snowtify CLI, its PATH entry, and the Frost layer.

Preserved Your other themes, extensions, and Spicetify configuration.
