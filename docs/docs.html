Guía de Snowtify

Documentación

Todo lo necesario para instalar, personalizar y recuperar Spotify desde PowerShell.

01

Instalación

Ejecuta PowerShell como usuario normal.

PowerShell
iwr -useb https://raw.githubusercontent.com/SnoW-099/snowtify/main/install.ps1 | iex

Snowtify reutiliza %APPDATA%\spicetify. Tus temas, extensiones y ajustes actuales siguen disponibles.

02

Primera ejecución

Crea una copia limpia y aplica tu personalización.

snowtify backup apply
03

Reparar Snowtify

Recupera la personalización cuando Spotify se actualiza o arranca mal.

snowtify repair
04

Actualizar

Descarga la última versión del canal independiente de Snowtify.

snowtify update
05

Marketplace

Instala temas y extensiones comunitarias desde Spotify.

Snowtify mantiene la API interna Spicetify.* y el alias de compatibilidad que Marketplace necesita.

06

Comandos esenciales

La referencia rápida para el uso diario.

ComandoUso
snowtify applyAplica la configuración actual.
snowtify repairRepara y reaplica tras una actualización.
snowtify updateActualiza Snowtify.
snowtify restoreDevuelve Spotify a su estado original.
snowtify configMuestra o modifica la configuración.
07

Problemas comunes

Soluciones rápidas antes de abrir una incidencia.

Snowtify desapareció después de actualizar Spotify

Ejecuta snowtify repair. Si Spotify cambió demasiado, actualiza primero con snowtify update.

Spotify abre con la interfaz bloqueada

Desactiva temporalmente el último tema o extensión y ejecuta snowtify apply.
